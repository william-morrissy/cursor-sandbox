# Cursor Demo Project

This is a demo environment for showcasing Cursor's key features and differentiators.

## Project Structure

```
cursor-sandbox/
├── backend/          # Python API with intentional issues for demo
├── frontend/         # React components and HTML demo
├── .cursor/          # Custom commands, rules, and prompts
└── docs/            # Documentation
```

## Getting Started

To reset this demo environment to baseline:
```bash
./reset-demo.sh
```

## Features Demonstrated

1. Multi-agent workflows
2. `.cursor` file operations
3. Review in one screen
4. Multiple agent results
5. Native browser integration
6. Git branch management
7. Code summarization

See `DEMO_WALKTHROUGH.md` for the complete demo script.
